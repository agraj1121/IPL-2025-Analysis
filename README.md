# IPL 2025 Batting and Bowling Performance Analysis

## Project Overview

This project analyzes the batting and bowling performances of players in IPL 2025 using Exploratory Data Analysis (EDA). The objective is to uncover patterns, compare player performances, identify impactful players, and generate meaningful insights using data visualization and statistical analysis.

The project focuses entirely on data analysis and storytelling without using Machine Learning, making it an ideal intermediate-level data science project.

---

## Objectives

* Analyze batting performances across IPL 2025.
* Analyze bowling performances and effectiveness.
* Compare players using various performance metrics.
* Identify consistent and impactful players.
* Discover relationships between different cricket statistics.
* Generate insights through data visualization.

---

## Dataset

The project uses two datasets:

* **Batters Dataset**
* **Bowlers Dataset**

The datasets contain player statistics such as runs, strike rates, wickets, economy rates, boundaries, matches played, and other performance metrics.

## Questions Answered

1. Who scored the most runs in IPL 2025?
2. Which bowlers took the most wickets?
3. Which batters had the highest strike rates?
4. Which bowlers had the best economy rates?
5. Is there a relationship between strike rate and runs scored?
6. Is economy rate related to wickets taken?
7. Which players are the most consistent?
8. Which teams contributed the most runs and wickets?
9. Which batters depend most on boundaries?
10. Which bowlers are both economical and wicket-taking?
11. Which batters combine high runs and high strike rates?
12. Who are the most impactful batters?
13. Who are the most impactful bowlers?
14. Who are the underrated players?
15. What would be the Best IPL 2025 Playing XI based on statistics?
